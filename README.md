# Portfolio

A React.js portfolio site inspired by the [Figma Portfolio – Community](https://www.figma.com/design/QLBjtu6SJEAt58gjui7YtO/Portfolio--Community-?node-id=0-1) template.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Customize

- **Hero:** Edit `src/components/Hero.jsx` for your name, tagline, and CTAs.
- **About:** Update `src/components/About.jsx` with your bio and location.
- **Skills:** Change the list in `src/components/Skills.jsx`.
- **Projects:** Edit the `projects` array in `src/components/Projects.jsx` and add image URLs if you like.
- **Contact:** Set your email and social links in `src/components/Contact.jsx`; wire the form to your backend or a service (e.g. Formspree) when ready.
